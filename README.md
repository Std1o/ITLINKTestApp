Соответствия требованиям:
1. Файл выкачивается при первом запросе с помощью ImagesCacheManager.
2. Наличие сети мониторится через NetworkMonitor и отправляется во флоу. При подписке, в случае, если список пуст и есть сеть, запускается повторный запрос.
3. Количество колонок считается в ColumnsCalculator.
4. Картинки сохраняются в ImageCache.
5. Перезагрузка картинок сделана в ReloadThumbnailUseCase.
6. Поддержка ночной темы есть (тема приложения зависит от темы устройства).

Также реализованы бонусные задачи на экране с оригиналами картинок.
